## Installation

Install with `yarn`:

```bash
$ yarn add wslib
```

Or, `npm` if you prefer:

```bash
$ npm install --save wslib
```

## Usage

Import the client and create a new instance:

```js
import { WebSocketClient } from 'wslib';
```
