## Installation

Install with `yarn`:

```bash
$ yarn add ws-client
```

Or, `npm` if you prefer:

```bash
$ npm install --save ws-client
```

## Usage

Import the client and create a new instance:

```js
import { WebSocketClient } from 'ws-client';
```
